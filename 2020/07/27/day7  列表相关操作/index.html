day7 列表相关操作

python 学习day7

01 列表相关操作

1. 数学运算： +， *

​ 列表1 + 列表2 —— 将列表1和列表2中的元素合并产生一个新的列表

​ 列表n / n列表 —— 将列表中的元素重复n次产生一个新的列表（n是正整数）

2. 比较运算

  • == 和 !=

    is 和 == 的区别：

    ​ == 判断的是值是否相等

    ​ is 判断的是地址是否相等

    补充： is 的用法

    ​

  • <, >, >=, >=

    列表1 > 列表2 —— 比较的是两个列表中第一组不相等的元素的大小

    3.相关函数

  • max、min

    max(序列) —— 获取列表中最大的元素

    min（列表）—— 获取列表中最小的元素

    注意：这儿的列表要求列表中的元素类型必须一致，并且支持比较运算

  • sorted

    sorted(列表) —— 将列表中的元素从小到大重新排序，产生一个新的列表

    sorted(列表， reverse = True) —— 将列表中的元素从大到小排序，产生一个新的列表

    注意：这儿的列表要求列表中的元素类型必须一致，并且支持比较运算

  • reversed

    reversed(列表) —— 将列表中的元素倒序产生一个新的迭代器对象

  • sun

    sun(列表) —— 计算列表中所有元素的和

    注意：序列中的元素必须全部是数字

  • list

    list(数据) —— 将数据转换成列表

    a. 所有容器型数据类型的数据都可以转换成列表

    b.将容器中的元素全部转换成列表中的元素

02 列表相关的方法

  • clear

    列表.clear() —— 清空列表

  • copy

    列表.copy() —— 复制列表中所有的元素产生一个新的列表

    列表。copy() == 列表[:] == 列表+[] == 列表*1 —— 浅拷贝

  • count

    列表.count(元素) —–统计指定元素在列表中出现的次数

  • extend

    列表.extend(序列) —— 将序列中的元素添加到列表中（不会产生新的列表）

  • index

    • 列表.index(元素) —— 获取指定元素在列表中的下标(正)

      a. 如果元素不存在会报错

      b. 如果元素有多个，只返回最前面那一个的下标值

    • 列表.index(元素，开始下标，结束下标) —— 获取元素在列表中的下标，查找范围在开始下标到结束下标选中的范围内

  • reverse

    列表.reverse() —— 将列表中的元素倒序（不会产生新的列表）

  • sort

    列表.sort()

    列表.sort(reverse= True)

05 列表推导式

1.列表推导式 —— 创建列表的表达式

  • 语法/格式1

    [表达式 for 变量 in 序列]

    让变量去序列中取值，每取一个值就创建列表中的一个元素，元素的值就是表达式的值

  • 语法2

    [表达式 for 变量 in 序列 if 条件语句]

    让变量取序列中值，每取一个值就判断条件语句是否为True，如果为True就创建一个列表元素，元素的值是表达式的值

  • 语法3

    [表达式 for 变量1 in 序列1 for 变量2 in 序列2 if 条件语句]
